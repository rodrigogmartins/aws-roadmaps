## üìö About the Exam

Below is a summary of the main information about the exam. These details and more can be found directly in the [**Exam Guide CLF-C02**](https://d1.awsstatic.com/onedam/marketing-channels/website/aws/en_US/certification/approved/pdfs/docs-cloud-practitioner/AWS-Certified-Cloud-Practitioner_Exam-Guide.pdf).

### What does the exam validate?

The exam validates the candidate‚Äôs ability to perform the following tasks:

- Explain the value of the AWS Cloud.

- Understand and explain the AWS Shared Responsibility Model.

- Understand the AWS Well-Architected Framework.

- Understand security best practices.

- Understand billing, economics, and cost management in the AWS Cloud.

- Describe and position the main AWS services, including compute, networking, database, and storage.

- Identify AWS services for common use cases.

### Recommended prior knowledge

This is an introductory AWS exam, so no prior experience with the AWS ecosystem is required.

### Content

The exam covers the following content domains and weightings:

- Domain 1: **Cloud Concepts** (24% of scored content)

- Domain 2: **Security and Compliance** (30% of scored content)

- Domain 3: **Cloud Technology and Services** (34% of scored content)

- Domain 4: **Billing, Pricing, and Support** (12% of scored content)
